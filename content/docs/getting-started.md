## Overview

AP0110 is an open-source project building the infrastructure for the Independent Internet (Web 4.0). This guide covers how to get involved as a developer, operator, or community contributor.

## For Developers

The AP0110 Network Operating System is modular and containerized. Contributions are welcome across:

- **Core networking stack** — protocol implementations and routing
- **Web interface** — the management dashboard and monitoring tools
- **Documentation** — guides, references, and translations

Visit the [GitHub organization](https://github.com/AP0110-Systems) to explore open issues and repositories.

## For Node Operators

Running an AP0110 node requires commodity hardware and a willingness to maintain uptime. Nodes contribute to mesh coverage and help route traffic for community members without reliable connectivity.

## For Community Partners

AP0110 works with schools, nonprofits, and local governments to deploy connectivity infrastructure. If your organization serves underserved communities, reach out via the Contribute page.
